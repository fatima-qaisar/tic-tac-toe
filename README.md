A responsive and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**, featuring dynamic gameplay, win/draw detection, and an animated winning line using the HTML5 Canvas.

---

## 🚀 Features

* 🎯 Two-player turn-based gameplay (X and O)
* 🧠 Win and draw detection logic
* ✨ Animated winning line using Canvas API
* 📱 Fully responsive design using viewport units
* 🔄 Reset and New Game functionality
* 🎨 Clean UI with hover effects
* 📐 Auto-resizing game board and canvas alignment

---

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling and responsiveness  
- **JavaScript (Vanilla)** – Game logic and DOM manipulation  
- **HTML5 Canvas API** – Drawing and animating the winning line

## 🎮 How to Play

1. Open the game in your browser.
2. Player **O** starts first.
3. Players take turns clicking empty cells.
4. The first player to align three marks in a row (horizontal, vertical, or diagonal) wins.
5. If all cells are filled without a winner, the game ends in a draw.
6. Use:

   * **Reset Game** → Restart the current game
   * **New Game** → Start a fresh game after completion
